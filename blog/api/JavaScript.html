# String 对象的方法

方法 描述
chartAt(index) 返回在指定位置的字符
charCodeAt(index) 返回在指定位置的字符 Unicode编码
fromCharCode(n1,n2,...,nX) 将Unicode编码转为字符
concat(str1, str2, ..., strX) 连接两个或多个字符串，并返回新的字符串
startsWith(str) 判断字符串是否已指定的子字符串开头
endsWith(str) 判断当前字符串是否是指定的子字符串结尾的(区分大小写)
indexOf(str, start) 返回指定的字符在字符串中首次出现的索引位置
lastIndexOf(str, start) 从后向前搜索字符串，并从起始位置(0)开始计算返回字符串最后出现的位置
includes(str, start) 查找字符串中是否包含指定的子字符串
replace(str/RegExp, newStr) 在字符串中查找匹配的子串，并替换与正则表达式匹配的子串
replaceAll(str/RegExp，newStr/function) 在字符串中查找匹配的子串，并替换与正则表达式匹配的所有子串
slice(start, end) 提取字符串的片段, 并在新的字符串中返回被提取的部分(复制片段为新字符串)
split(str/RegExp, limit) 把字符串分割为字符串数组
substr(start, length) 从起始索引号提取字符串中指定长度(start, length)的字符
substring(from, to) 提取字符串中两个指定的索引号之间(start, end)的字符
trim() 去除字符串两边的空白
toString() 返回一个字符串
search(str/RegExp) 查找与正则表达式相匹配的值 返回值 Number
match(RegExp) 查找找到一个或多个正则表达式的匹配 返回值 Array
repeat(count) 复制字符串指定次数，并将它们连接在一起返回
toLowerCase() 把字符串转换为小写
toUpperCase() 把字符串转换为大写
toLocaleLowerCase() 根据本地主机的语言环境把字符串转换为小写
toLocaleUpperCase() 根据本地主机的语言环境把字符串转换为小写
valueOf() 返回某个字符串对象的原始值
// 返回字符串第一个和最后一个字符
let str = 'Hello world';

// 拼接字符串
const str1 = 'Hello ';

const str3 = ' Have a nice day!';

const str1 = "Hello world, welcome to the Runoob.";;

function is_wechat() {
    var ua = navigator.userAgent.toLowerCase();
    return ua.match(/MicroMessenger/i)=="micromessenger" ? true: false;

// 大小写敏感的查找
let str = "Mr. Blue has a blue house";
let index = str.search('blue'); // 返回 15

// 忽略大小写的检索
let str = "Mr. Blue has a blue house";

// 全局替换字符串
let str = "Mr Blue has a blue house and a blue car";

// 全局替换 忽略大小写

let n = str.replace(/blue/gi, "red"); // 返回：'Mr red has a red house and a red car'

// 为String添加自定义方法
String.prototype.replaceAll = function(search, replacement) {
    let target = this;
    return target.replace(new RegExp(search, 'g'), replacement);
};

// 分割字符串数组
let str = "1 2 3 4 5";

// 常和Array join() 配套使用

// 使用limit 分割(该参数可指定返回的数组的最大长度)

// substr(start, length) 截取指定数目的字符串
let str = 'Hello world';
let n = str.substr(2, 3); // 从索引2开始截取3个字符， 返回：'llo'

// substr(start) length 可选 不填就到结束
let str = "Hello world!";
let n = str.substr(2); // 从索引2开始截取到最后, 返回：'llo world!'

// substring(start, end); 截取两个位置之间的字符
let str = 'Hello world';
let n = str.substring(2, 3); // 截取索引从2到3的字符， 返回：'l'

// substring(start); end 可选 不填就到结束
let str = "Hello world!";
let n = str.substring(2); // 从索引2开始截取到最后， 返回：'llo world!'

// 去除头尾空格
let str = '      Run   ';
let n = str.trim(); // 返回: 'Run'

// 自定义方法去除空格
function myTrim(str) {
    return str.replace(/^\s+|\s+$/gm,'');
}
let n = myTrim('   Run '); // 返回: 'Run'

# Array 数组的方法

方法 描述
concat(arr2,...,arrX) 连接两个或多个数组, 并返回新数组，原有数组不改变
fill(value,start,end) 使用一个固定值来填充数组
from(obj,fun,val) 通过给定的对象中创建一个数组。
copyWithin(target,start,end) 从数组的指定位置拷贝元素到数组的另一个指定位置中
shift() 删除并返回数组的第一个元素。
pop() 删除数组的最后一个元素并返回删除的元素。
unshift(item1,...,itemX) 向数组的开头添加一个或更多元素，并返回新的长度。
push(item1,...,itemX) 向数组的末尾添加一个或更多元素，并返回数组新的长度。
find(fun(currVal,index,arr), thisVal) 返回符合传入测试（函数）条件的数组元素。
findIndex(fun(currVal,index,arr), thisVal) 返回符合传入测试（函数）条件的数组元素索引。
includes(searchEl, fromIndex) 判断一个数组是否包含一个指定的值。
indexOf(item,start) 搜索数组中的元素，并返回它所在的位置。
lastIndexOf(item,start) 搜索数组中的元素，并返回它最后出现的位置。
join(separator) 把数组的所有元素放入一个字符串。
entries() 返回数组的可迭代对象。
keys() 返回数组的可迭代对象，包含原始数组的键(key)。
forEach(callback(currVal,index,arr), thisVal) 数组每个元素都执行一次回调函数。
every(fun(currVal,index,arr), thisVal) 检测数值元素的每个元素是否都符合条件。
filter(fun(currVal,index,arr), thisVal) 检测数值元素，并返回符合条件所有元素的数组。
map(fun(currVal,index,arr),thisVal) 通过指定函数处理数组的每个元素，并返回处理后的数组。
reduce(fun(total,currVal,currIndex,arr),initVal) 将数组元素计算为一个值（从左到右）。
reduceRight(fun(total,currVal,currIndex,arr),initVal) 将数组元素计算为一个值（从右到左）。
reverse() 反转数组的元素顺序。
some(fun(currVal,index,arr), thisVal) 检测数组元素中是否有元素符合指定条件。
sort(sortfun(a,b)) 对数组的元素进行排序。
slice(start,end) 选取数组的一部分，复制并返回一个新数组。
splice(index,howmany,item1,...,itemX) 从数组中添加或删除元素。参数howmany为(要删除多少个元素,可0), 返回已删除的元素数组
toString() 把数组转换为字符串，并返回结果。
valueOf() 返回数组对象的原始值。
Array.isArray(obj) 判断对象是否为数组。
Array.of(element0,...elementN) 将一组值转换为数组。
Array.at(index) 用于接收一个整数值并返回该索引对应的元素，允许正数和负数。负整数从数组中的最后一个元素开始倒数。
Array.flat(depth) 创建一个新数组，这个新数组由原数组中的每个元素都调用一次提供的函数后的返回值组成。
Array.flatMap(callback(currVal,index,arr), thisArg) 使用映射函数映射每个元素，然后将结果压缩成一个新数组。
// 拼接多个数组
let arr1 = ['a', 'b', 'c'];
let arr2 = ['d', 'e', 'f'];
let arr3 = ['g', 'h', 'i'];
let arr = arr1.concat(arr2, arr3); // 返回：['a', 'b', 'c', 'd', 'e', 'f', 'g', 'h', 'i']

// .copyWithin(target, start, end) 复制数组的前两个元素到第三和第四个位置上
let arr = ['A', 'B', 'C', 'D', 'E', 'F', 'G'];
arr.copyWithin(2, 0, 2); // arr修改为： ['A', 'B', 'A', 'B', 'E', 'F', 'G']

// 填充A到最后两个元素
let arr = ['B', 'C', 'D', 'E'];
arr.fill('A', 2, 4); //  arr修改为：['B', 'C', 'A', 'A']

// 创建一个2X3并且默认为0的二维数组
let dp = Array.from({length: 2}, () => new Array(3).fill(0));
// 返回 [[0,0,0],[0,0,0]]

// push(item,...) 向数组尾部添加元素
let arr = [];
arr.push('A');
arr.push('B', 'C'); // arr改变为: ['A', 'B', 'C']

// pop() 从数组尾部删除最后一个元素
let last = arr.pop(); // 返回: 'C', arr改变为: ['A', 'B']

// unshift(item,...) 向数组头部添加元素
let arr = ['A', 'B'];
let len = arr.unshift('D'); // 返回len: 3, arr改变为: ['D', 'A', 'B']

// shift() 从数组头部删除第一个元素
let arr = ['D', 'A', 'B'];
let first = arr.shift(); // 返回: 'D', arr改变为: ['A', 'B']

// slice(start, end) 截取片段，复制新数组
let arr = ['A', 'B', 'C', 'D', 'E'];
let copy = arr.slice(-3,-1); // 截取倒数第3个(含),到倒数第1个(不含)的两个元素, 返回: ['C', 'D']
let last = arr.slice(-3); // 截取最后三个元素 返回：['C', 'D', 'E']

// splice(index, howmany, item1,...itemX) 添加或删除元素
let arr = ['A', 'B'];
arr.splice(2, 0, 'C', 'D'); // 从索引2开始，添加新元素，返回已删除[], arr改变为 ['A', 'B', 'C', 'D']
arr.splice(2, 1, 'E', 'F'); // 移除数组第三个元素, 并在数组第3个位置添加新元素'E','F', 返回已删除['C'], arr改变为:['A', 'B', 'E', 'F', 'D'] 
arr.splice(2, 2); // 从第3个位置开始删除2个元素; 返回已删除['E', 'F'], arr改变为：['A', 'B', 'D']

// 移除数组指定元素
const index = arr.indexOf('B');
if (index != -1) {
    arr.splice(index, 1);
}

// sort((a,b) => {}) 数组排序
let arr = [1,8,3,4,2,10,3,5,1];
arr.sort((a, b) => {return a - b}); // 数组按升序排序

// 对象按属性排序
let arr = [{id: 'A', size: 2, age: 10}, {id: 'B', size: 1, age: 18}];
arr.sort((a, b) => a.size - b.size) // 按size 升序

const map = new Map();

if (map.has('B')) {
    map.delete('B');
}

for (let key of map.keys()) {}
for (let value of map.values()) {}
for (let [key, value] of map.entries()) {}
map.forEach((value, key) => { })

map.clear();

# Map 对象的方法

方法 描述
set(key, value) 添加或修改元素
get(key) 获取元素
has(key) 是否存在元素
delete(key) 是否包含元素
values() 获取所有值得迭代对象
keys() 获取所有key得迭代对象
entries() 获取所有[key,value]得迭代对象
clear() 清空map
forEach((value, key) {}) 遍历
// Map 转为数组
let map = new Map();
map.set('A', '1');
map.set('B', '2');

let arr = [...map]; // [['A', '1'], ['B', '2']]
let arr1 = Array.from(map); // [['A', '1'], ['B', '2']]

// Map 转为Object
let obj = {};
map.forEach((v, k) => obj[k] = v); // {A: '1', B: '2'}
// Map 转为JSON
JSON.stringify(obj); // '{"A":"1","B":"2"}'

// Object转Map
let map = new Map(Object.entries(obj));

// 数组转Map
let arr = [
    ['A', '1'],
    ['B', '2']
]
let map = new Map(arr);

// Map与Object的区别
1.Object中，键必须是简单类型(整数，字符串或符号), Map中可以任何类型(对象，数组等)
2.Map中元素顺序有序，即插入顺序
3.继承,Map是Object的一个实例,

队尾(rear)：允许插入的一端, 指向队列最后一个元素

# Queue 队列的方法

方法 描述
offer() 入队，从队尾添加元素， 如果失败，返回false, JS里用 queue.push()
add() 入队，从队尾添加元素, 如果失败抛出异常, JS里用 queue.push()
poll() 出队，从队头移除元素, 如果此队列为空, 返回 null, JS里用 queue.shift()
remove() 出队，从队头移除元素, 如果此队列为空, 抛出异常, JS里用 queue.shift()
peek() 取队头第一个元素, 获取但不移除，JS里用 queue[0]
isEmpty() 是否空栈，JS里用 queue.length === 0
size() 队列里有多少元素, JS里用 queue.length
clear() 清空队列, JS里用 queue = []
